# Project Overview

## Introduction

This project presents the planning of a CRM system implementation using Microsoft Project and ERP concepts based on Comarch ERP Optima.

It demonstrates the complete project planning process, including scheduling, resource allocation, risk assessment, cost estimation, and business process analysis.

---

## Project Scope

The project covers:

- Work Breakdown Structure (WBS)
- Project scheduling
- Resource planning
- Risk analysis
- Cost estimation
- Critical Path analysis
- ERP business process modelling

---

## Project Outcome

The final result is a complete implementation plan prepared in Microsoft Project together with supporting documentation describing the project lifecycle and ERP-related business processes.

The project demonstrates practical skills in project planning and ERP implementation concepts.
