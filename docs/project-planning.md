# Project Overview

## Introduction

This repository presents the planning and documentation of a CRM system implementation project based on **Comarch ERP Optima**. The project was developed using **Microsoft Project** to simulate a real-world ERP implementation process, covering project planning, scheduling, resource management, risk assessment, and business process analysis.

---

## Objectives

The main objectives of this project were to:

- plan a CRM implementation project using project management best practices,
- create a Work Breakdown Structure (WBS) and project schedule,
- allocate and analyse project resources,
- identify implementation risks and estimate project costs,
- model an ERP-supported business process.

---

## Project Scope

The project includes:

- project planning and scheduling,
- resource management,
- risk and cost analysis,
- Critical Path analysis,
- ERP business process modelling,
- project documentation.

---

## Technologies & Tools

- Microsoft Project
- Comarch ERP Optima
- Microsoft Office
- Markdown
- Git & GitHub

---

## Project Outcome

The result of this project is a complete implementation plan demonstrating the key stages of an ERP/CRM deployment. It highlights practical skills in project management, ERP implementation planning, documentation, and business process analysis.
