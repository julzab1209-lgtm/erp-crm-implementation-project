# Lessons Learned

## Overview

This project provided practical experience in planning an ERP implementation using **Microsoft Project** and **Comarch ERP Optima**. It combined project management principles with ERP concepts commonly used in business environments.

---

## Skills Developed

During this project, I improved my skills in:

- project planning and scheduling,
- Work Breakdown Structure (WBS) creation,
- resource allocation,
- project risk assessment,
- cost estimation,
- Critical Path analysis,
- ERP business process modelling,
- technical documentation using Markdown and GitHub.

---

## Challenges

The main challenges included:

- organizing project activities into a realistic implementation plan,
- balancing project resources,
- identifying potential implementation risks,
- creating clear and structured project documentation.

---

## Professional Takeaways

This project strengthened my understanding of ERP implementation planning and project management practices. It also improved my ability to document technical projects and present them in a professional portfolio.
